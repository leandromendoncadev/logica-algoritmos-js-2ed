# Lógica de Programação com JavaScript (2ª ed.) — Exercícios e Estudos

Este repositório registra meus estudos e minha prática de programação durante a leitura do livro 
**“Lógica de Programação e Algoritmos com JavaScript – 2ª Edição”**, de **Edécio Fernando Iepsen**.

A proposta é resolver os exercícios **um a um**, fazendo commits frequentes, e no final ter uma estrutura de arquivos semelhante ao material de apoio disponibilizado (pastas, imagens e arquivos usados nos exercícios).

---

## Como este repositório está organizado

A estrutura segue o formato por capítulos (ex.: `cap01`, `cap02`, ...), compatível com o pacote de apoio disponibilizado pela editora/autor:

- `cap01/`
- `cap02/`
- `cap03/`
- ...
- `cap10/`

> Observação: alguns arquivos (imagens/HTML/CSS/JS) são utilizados pelos exercícios para exibição/entrada de dados no navegador.
